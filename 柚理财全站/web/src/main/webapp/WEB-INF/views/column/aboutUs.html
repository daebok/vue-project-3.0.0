京圆柚理财是中国（浙江）领先的互联网金融信息服务平台，运营公司主体为浙江邦理投资管理有限公司，于2015年12月1日正式上线，2015年12月正式与新浪签订第三方支付协议，实现资金全程由新浪支付托管。2016年4月通过阿里金融云微金融认证，成功接入阿里金融云。

截止至2017年2月，平台用户数突破70,000人，累积投资金额突破2.5亿元，已为海量用户提供财富升值服务。平台专注于互联网金融信息服务，利用互联网大数据为广大用户提供专业高效的普惠金融服务，致力于成为用户首选的安全可靠的理财平台。

自成立以来，京圆柚理财始终秉持着“安全创造财富、互联传递幸福”的价值观，以专业化水准为客户服务，其团队成员由专业金融机构、法律行业、经济研究和电子商务等领域的专业人士组成。
